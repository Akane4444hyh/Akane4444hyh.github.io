交易與市場基礎

  • 標的資產（Asset）：實際交易的對象，如股票、ETF、期貨、期權
  • 多頭/空頭（Long/Short）：多頭是先買後賣期待上漲；空頭是先賣後買期待下跌（在A股，個人投資者通常需通過融券、股指期貨等工具實現）
  • 成交價/開盤價/收盤價：成交價：某一筆交易實際發生的價格；開盤價：當日第一筆成交價格，反映隔夜信息的集中釋放；收盤價：在連續競價市場（如A股），通常指收盤集合競價產生的價格。因其避免尾盤操縱、穩定可復現，是量化研究中最常用的基準價格。
  • 滑點（Slippage）：預期成交價與實際成交價的差額，常由市場波動、流動性不足或訂單規模導致
  • 衝擊成本（Impact Cost）：大額訂單本身對市場價格產生的不利影響，是滑點的主要來源之一
  • 手續費（Commission）：每次交易產生的固定或按比例收取的成本，高頻策略必須重點考慮
  • 流動性（Liquidity）：資產是否能被快速、低成本地買賣成交
  • 成交量/成交額（Volume/Turnover）：成交量指成交的股數/手數；成交額指成交的金額，反映市場活躍度（另見“換手率”）
  • 盤口/訂單簿（Order Book）：顯示當前買賣掛單價格和數量的實時列表
  • 最小變動單位（Tick Size）：價格變動的最小單位，如A股爲0.01元
  • 做市商（Market Maker）：提供雙向報價，爲市場提供流動性的機構

收益與風險指標

  • 收益率（Return）：投資盈利的比例，可分爲絕對收益和相對收益
  • 回撤（Drawdown）：從歷史最高點到後續最低點的跌幅；最大回撤表示歷史最慘的一次虧損
  • 波動率（Volatility）：收益率波動的標準差，是衡量價格不確定性的核心風險指標
  • 夏普比率（Sharpe Ratio）：每承擔一單位總風險獲得的超額收益，越高說明風險調整後收益越好
  • 信息比率（Information Ratio）：超額收益與跟蹤誤差的比值，衡量單位主動風險所帶來的超額收益，用於評估主動管理能力的效率與穩定性
  • 年化收益/年化波動率：將不同周期數據標準化爲年率，便於策略橫向比較
  • 索提諾比率（Sortino Ratio）：類似夏普比率，但只考慮下行風險

策略思想與類型

  • 基本面分析（Fundamental）：研究公司內在價值，如財務數據、行業地位、管理團隊
  • 技術分析（Technical）：基於價格、成交量等市場數據進行分析，如K線、均線、指標
  • 因子（Factor）：可解釋股票收益的特徵變量，如價值、規模、動量、質量因子
  • 動量（Momentum）：漲得多的資產可能繼續上漲，跌得多的可能繼續下跌
  • 均值回歸（Mean Reversion）：價格偏離其長期均值後會回歸到平均水平
  • 事件驅動（Event Driven）：利用財報發布、並購公告、政策變化等事件進行交易
  • 市場中性（Market Neutral）：通過同時構建多空頭寸對衝市場方向性風險
  • 統計套利（Statistical Arbitrage）：基於資產價格間長期存在的穩定統計關系（如協整、價差均值回歸） 進行交易。當價差偏離時構建多空組合，期待其回歸並獲利。
  • 高頻交易（HFT）：利用極短時間內的市場微觀結構進行交易

回測與驗證

  • 回測（Backtest）：使用歷史數據驗證策略邏輯和效果
  • 過擬合（Overfitting）：策略過度優化，只適合歷史數據而不具備未來泛化能力
  • 樣本外測試（Out-of-sample）：使用未參與參數優化的數據驗證策略
  • 幸存者偏差（Survivorship Bias）：只使用當前仍存在的標的數據回測，導致結果虛高
  • 前視偏差（Look-ahead Bias）：回測中誤用了當時不可得的未來信息
  • 交易成本：回測中常被低估的實際成本，包括手續費、滑點和衝擊成本

風險控制與資金管理

  • 倉位（Position）：在某一標的上的投資資金佔總資金的比例
  • 滿倉/半倉/空倉：滿倉表示資金全部投入；半倉保留部分現金；空倉無持倉
  • 止損/止盈（Stop Loss/Take Profit）：止損用於控制虧損，止盈用於鎖定收益
  • 換手率（Turnover）：單位時間內資產的交易頻率。
  • 風險預算（Risk Budgeting）：爲不同資產或策略分配可承受的風險額度
  • VaR/CVaR：VaR爲一定置信度下的最大可能損失；CVaR爲超過VaR的極端損失平均值
  • 槓杆（Leverage）：通過借入資金或衍生品放大投資頭寸，從而同步放大收益與風險
  • 分散化（Diversification）：通過投資不同資產降低整體風險

基準與績效評估

  • 基準（Benchmark）：用於對比策略表現的參考標準，如滬深300、中證500
  • Alpha（α）：策略相對於基準的超額收益，是主動管理的核心目標
  • Beta（β）：策略收益對基準收益波動的敏感度，衡量市場風險暴露
  • 跟蹤誤差（Tracking Error）：策略收益與基準收益差異的波動率
  • 最大回撤期（Max Drawdown Duration）：從回撤開始到恢復新高所需的最長時間

信號與執行

  • 信號（Signal）：指示買賣方向、時機或數量的量化信息
  • 執行（Execution）：將交易指令轉化爲實際成交的過程
  • 市價單/限價單：市價單保證成交但不保證價格；限價單保證價格但不保證成交
  • 算法交易（Algorithmic Trading）：使用程序自動拆分和執行大額訂單以降低衝擊成本
  • 執行落差（Implementation Shortfall）：策略理論收益與實際執行收益的差距

市場與參與者

  • 一級/二級市場：一級市場是發行市場；二級市場是流通交易市場
  • 交易所/場外市場：交易所是集中競價場所；場外市場是雙邊協商交易
  • 機構投資者/散戶：機構是專業投資者；散戶是個人投資者
  • 量化基金/對衝基金：主要使用量化方法的投資機構
  • 自營交易公司：使用自有資金進行交易的公司
  • 養老金/保險資金：長期機構投資者，追求穩健收益

12.19

  模拟练习（均线+RSI策略回测）

import akshare as ak
import pandas as pd
import matplotlib.pyplot as plt

# -------------------------------
# 1. 获取数据（贵州茅台）
# -------------------------------
df = ak.stock_zh_a_hist(symbol="600519", period="daily",
                        start_date="20210101", end_date="20251130",
                        adjust="qfq")

df['日期'] = pd.to_datetime(df['日期'])
df.set_index('日期', inplace=True)
df.sort_index(inplace=True)

# -------------------------------
# 2. 均线计算
# -------------------------------
df['MA10'] = df['收盘'].rolling(10).mean()
df['MA60'] = df['收盘'].rolling(60).mean()

# -------------------------------
# 3. RSI 指标计算
# -------------------------------
def rsi(prices, period=14):
    delta = prices.diff()
    gain = delta.clip(lower=0)
    loss = -delta.clip(upper=0)
    avg_gain = gain.rolling(period).mean()
    avg_loss = loss.rolling(period).mean()
    rs = avg_gain / avg_loss
    return 100 - 100 / (1 + rs)

df['RSI'] = rsi(df['收盘'])

# -------------------------------
# 4. 数据清洗
# -------------------------------
df = df[df['收盘'] > 0]
df = df.dropna(subset=['MA10', 'MA60', 'RSI'])

# -------------------------------
# 5. 生成交易信号
# -------------------------------
# 先生成原始信号（基于当天收盘后的数据）
df['Signal_Raw'] = 0

# 买入条件：MA10>MA60 且 RSI<30
# 趋势向上（短期均线在长期均线上方）且超卖（RSI低于30）
df.loc[(df['MA10'] > df['MA60']) & (df['RSI'] < 30), 'Signal_Raw'] = 1

# 卖出条件：MA10<MA60 或 RSI>70
# 趋势向下或超买，认为应该卖出
df.loc[(df['MA10'] < df['MA60']) | (df['RSI'] > 70), 'Signal_Raw'] = -1

# 信号延迟一天执行，避免未来函数
# 原理：今天收盘后才能计算出今天的RSI和均线
#      所以今天的信号只能在明天执行
#      shift(1)表示将信号向后移动一天
# 例如：12月1日收盘后生成买入信号，12月2日才能买入
df['Signal'] = df['Signal_Raw'].shift(1)

# 删除Signal为NaN的第一行（因为shift后第一行没有前一天的信号）
df = df.dropna(subset=['Signal'])

# -------------------------------
# 6. 回测参数初始化
# -------------------------------
initial_cash = 10000  # 初始资金（单位：元），回测起始时的本金
cash = initial_cash   # 当前现金余额，随着买卖操作动态变化
shares = 0            # 当前持有的股票数量（股数），0表示空仓
portfolio = []        # 每日总资产记录列表，用于绘制净值曲线和计算收益率
buy_points = []       # 买入点记录列表，格式：[(日期, 价格), ...]，用于绘图标记
sell_points = []      # 卖出点记录列表，格式：[(日期, 价格), ...]，用于绘图标记
min_holding_days = 5  # 最小持仓天数，避免频繁交易（减少交易成本和噪音信号）
holding_days = 0      # 当前持仓天数计数器，每天持仓时+1，卖出后归零

# -------------------------------
# 7. 回测循环
# -------------------------------
for date, row in df.iterrows():

    price = row['开盘']  # 使用当天开盘价进行买卖操作
    signal = row['Signal']  # 这是前一天收盘后生成的信号
    
    # 如果持仓，持仓天数 +1
    if shares > 0:
        holding_days += 1

    # ---------------------------
    # 买入逻辑
    # ---------------------------
    # 条件：收到买入信号(signal=1) 且 当前空仓(shares=0)
    if signal == 1 and shares == 0:
        shares = cash / price  # 全仓买入，计算可买股数
        cash = 0               # 现金置 0（全仓操作）
        buy_points.append((date, price))  # 记录买入点，用于后续绘图
        holding_days = 0       # 重置持仓天数计数器
        print(f"买入: {date.date()} @ {price:.2f}")

    # ---------------------------
    # 卖出逻辑（持仓超过最短周期）
    # ---------------------------
    # 条件：收到卖出信号 且 当前有持仓 且 持仓天数>=最小持仓天数
    # min_holding_days 用于避免频繁交易
    elif signal == -1 and shares > 0 and holding_days >= min_holding_days:
        cash = shares * price     # 卖出所有股票，更新现金
        shares = 0                # 持仓置 0
        sell_points.append((date, price))  # 记录卖出点
        holding_days = 0           # 重置持仓天数
        print(f"卖出: {date.date()} @ {price:.2f}")

    # ---------------------------
    # 总资产计算
    # ---------------------------
    # 每天记录总资产 = 现金 + 持仓股数 × 当前价格
    # 用于绘制净值曲线和计算收益率
    portfolio.append(cash + shares * price)

# 保存总资产曲线到DataFrame
df['Portfolio'] = portfolio

# -------------------------------
# 8. 绘图
# -------------------------------
plt.figure(figsize=(12, 8))
plt.rcParams['font.sans-serif'] = ['Microsoft YaHei']  # 设置中文字体
plt.rcParams['axes.unicode_minus'] = False              # 负号正常显示

# ---------------------------
# 子图1：价格 + 均线 + 买卖点
# ---------------------------
plt.subplot(3,1,1)
plt.plot(df.index, df['收盘'], label='收盘价', linewidth=1.5)
plt.plot(df.index, df['MA10'], label='10日均线', color='orange', linewidth=1)
plt.plot(df.index, df['MA60'], label='60日均线', color='red', linewidth=1)
# 绘制买入点（绿色向上三角形）
if buy_points:
    dates, prices = zip(*buy_points)
    plt.scatter(dates, prices, marker='^', color='green', s=100, label='买入', zorder=5)
# 绘制卖出点（红色向下三角形）
if sell_points:
    dates, prices = zip(*sell_points)
    plt.scatter(dates, prices, marker='v', color='red', s=100, label='卖出', zorder=5)
plt.title('贵州茅台')
plt.legend()
plt.grid(True, alpha=0.3)

# ---------------------------
# 子图2：净值对比
# ---------------------------
plt.subplot(3,1,2)
# 策略净值曲线（我们的策略表现）
plt.plot(df.index, df['Portfolio'], label='策略净值', color='darkblue', linewidth=2)
# 买入持有基准（如果一开始买入并一直持有）
baseline = initial_cash * df['收盘'] / df['收盘'].iloc[0]
plt.plot(df.index, baseline, label='买入持有', linestyle='--', color='gray', linewidth=2)
plt.title('净值对比')
plt.legend()
plt.grid(True, alpha=0.3)

# ---------------------------
# 子图3：RSI指标
# ---------------------------
plt.subplot(3,1,3)
plt.plot(df.index, df['RSI'], label='RSI', color='purple', linewidth=1)
# RSI超卖线（30以下认为超卖，可能反弹）
plt.axhline(y=30, color='green', linestyle='--', alpha=0.5, label='超卖线(30)')
# RSI超买线（70以上认为超买，可能回调）
plt.axhline(y=70, color='red', linestyle='--', alpha=0.5, label='超买线(70)')
plt.title('RSI指标')
plt.legend()
plt.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

# -------------------------------
# 9. 回测结果统计
# -------------------------------
final_value = df['Portfolio'].iloc[-1]             # 最终总资产
total_return = (final_value / initial_cash - 1) * 100   # 策略总收益率
baseline_return = (df['收盘'].iloc[-1] / df['收盘'].iloc[0] - 1) * 100  # 买入持有收益率

# 计算最大回撤
# 最大回撤 = 从历史最高点到后续最低点的最大跌幅
df['Cummax'] = df['Portfolio'].cummax()  # 累计最高净值
df['Drawdown'] = (df['Portfolio'] / df['Cummax'] - 1) * 100  # 每日回撤百分比
max_drawdown = df['Drawdown'].min()  # 最大回撤值（负数，越小风险越大）

print("=" * 60)
print(f"初始资金:        ¥{initial_cash:,.0f}")
print(f"最终资金:        ¥{final_value:,.2f}")
print(f"策略总收益率:    {total_return:.2f}%")
print(f"买入持有收益率:  {baseline_return:.2f}%")
print(f"超额收益:        {total_return - baseline_return:.2f}%")
print(f"最大回撤:        {max_drawdown:.2f}%")
print(f"交易次数:        买入{len(buy_points)}次, 卖出{len(sell_points)}次")
print("=" * 60)
